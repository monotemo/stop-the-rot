# Stop The Rot

A free "earn your screen time" app for kids.

Kids complete chores to earn coins, then spend coins on screen time packages. Integrated with iOS Shortcuts for Screen Time control.

## Coin Values
- Easy chore: 5 coins
- Medium chore: 10 coins
- Hard chore: 20 coins

## Screen Time Packages
- 15 min = 5 coins
- 30 min = 10 coins
- 1 hour = 15 coins
- 2 hours = 25 coins

## Development

```bash
npm install
npm run dev
```
